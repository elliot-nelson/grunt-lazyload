# grunt-lazyload

Gruntplugin Lazy Loading

## Getting Started
Install the module with: `npm install grunt-lazyload`

```javascript
var lazyload = require('grunt-lazyload').lazyload(grunt);
lazyload('grunt-contrib-jshint', 'jshint');
```
